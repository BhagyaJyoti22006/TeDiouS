1. Create a new Repository in Github.

2. Copy the `email.json` file from current directory to the repo directory inside `<repo>\`

3. Replace `<roll>` with the correct roll number.

4. Push the Respository to `main`.

5. Go to Github Repository and click on `email.json` and then click on `Raw`.

6. Copy the raw Github URL which would look like `https://raw.githubusercontent.com/<user>/<repo>/refs/heads/main/email.json`.
