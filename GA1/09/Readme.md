1. Create a new Repository in Github.

2. Copy the `index.html` file from current directory to the repo directory inside `<repo>\`.

3. Replace `<roll>` with the correct roll number.

4. Push the Respository to `main`.

5. Go to Github Repository and go to `Settings` and click on `Pages` from left sidebar.

6. Under `Build and deployment` select `source` as `Deploy from a branch` and `branch` as `main` and `folder` as `/(root)`.

7. Click on the `Save` button and wait for GitHub to publish the pages.

8. Goto the Repository URL. On right side, under `Deployments`, select `github-pages`. 

9. Copy the displayed URL which would look like `https://<user>.github.io/<repo>/`.
