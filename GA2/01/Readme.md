1. Open `q-fastapi-metrics-cors-server/index.py` and replace `<roll>` with the correct roll number and `<ALLOWED_CORS_ORIGIN>` with the assigned values.

2. Open terminal in the directory `q-fastapi-metrics-cors-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-fastapi-metrics-cors-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-fastapi-metrics-cors-s.vercel.app`.
