1. Open `q-oauth-jwks-verify-server/index.py` and replace `<ISS>`, `<AUD>` and `<RS256_PUBLIC_KEY>` with the assigned values.

2. Open terminal in the directory `q-oauth-jwks-verify-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-oauth-jwks-verify-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-oauth-jwks-verify-serv.vercel.app`.
