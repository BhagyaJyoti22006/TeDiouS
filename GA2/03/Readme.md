1. Open `q-config-precedence-server/index.py` and replace `<DEFAULTS>`, `<CONFIG_YAML>`, `<ENV_FILE>` and `<OS_ENV_VARS>` with the assigned values.

2. Open terminal in the directory `q-config-precedence-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-config-precedence-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-config-precedence-serv.vercel.app`.
