1. Open `q-deploy-analytics-platform-server/index.py` and replace `<roll>` with the correct roll number and `<API_KEY>` with the assigned values.

2. Open terminal in the directory `q-deploy-analytics-platform-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-deploy-analytics-platform-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-deploy-analytics-platf.vercel.app`.
