1. Open `q-observability-metrics-server/index.py` and replace `<roll>` with the correct roll number.

2. Open terminal in the directory `q-observability-metrics-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-observability-metrics-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-observability-metrics.vercel.app`.
