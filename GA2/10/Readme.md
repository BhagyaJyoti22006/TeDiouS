1. Open `q-middleware-ratelimit-cors-server/index.py` and replace `<roll>` with the correct roll number and `<ALLOWED_CORS_ORIGIN>`, `<REQUESTS>` and `<BUCKETS>` with the assigned values.

2. Open terminal in the directory `q-middleware-ratelimit-cors-server`.

3. Run following command:  
`vercel --prod`

4. Follow the instructions to publish the project. Choose a name like `<roll>-q-middleware-ratelimit-cors-server`. 

5. Copy the `Aliased` URL which would be like `https://<roll>-q-middleware-ratelimit-c.vercel.app`.
